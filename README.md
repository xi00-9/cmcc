# Developer Tools Collection

## Products

### 1. Webhook Debugger ($19)
Real-time HTTP request capture and replay. `npm start` → get your webhook URL.
**Features:** All HTTP methods, SSE real-time updates, one-click replay, GitHub-dark UI.

### 2. API Inspector ($29)  
Advanced request inspector with custom response simulation.
**Features:** Custom status codes, response delays, JSON responses via query params.

### 3. DevUtils CLI ($9)
Command-line developer utilities: JSON fmt, JWT decode, base64, UUID, timestamp, port check, SHA256.
```bash
npm install -g ./devutils-cli
devutils uuid
```

### 4. Stock Alert ($14)
Real-time A-stock price monitor with desktop alerts for price breakthroughs.
```bash
python stock_alert.py
```
